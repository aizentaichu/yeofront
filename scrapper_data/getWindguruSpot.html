Afficher la carte
Cochez la case "Afficher la carte" pour afficher les spots sur la carte Google. La carte montrera toujours les spots visibles dans le menu de choix de spot.

Par exemple: si vous choisissez Italie, tous les spots d'Italie apparaîtront dans le menu et sur la carte. Si vous voulez afficher tous les spots en Europe choisissez "TOUT" dans le menu pays. Vous pouvez même afficher TOUS les spots disponibles en choisissant TOUT comme continent puis TOUT comme pays, mais attention cela chargera beaucoup de spots et sera lent...

Internet Explorer n'est pas recommandé pour la carte des spots, il est très lent comparé aux autres navigateurs récents. Les meilleurs choix sont Firefox, Google Chrome, Opera ou Safari.

Pour utiliser un fichier .kmz::

  1. télécharger et installer Google Earth
  2. cliquez le lien .kmz
  3. ouvrez le fichier directement du navigateur ou si ça ne marche pas, choisissez "Enregistrer sous" et ouvrez le fichier par Google Earth
